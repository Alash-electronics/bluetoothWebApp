GITHUB ABOUT SECTION
====================

Краткое описание (Description):
-------------------------------
Universal web-based Bluetooth controller for Arduino and embedded devices. Supports HC-06, HC-05, HM-10, ESP32 and other BLE/Bluetooth modules. Multiple control modes: Terminal, RC Car, Joystick, Smart Home. Built with React + TypeScript.

Альтернативный вариант на русском:
-----------------------------------
Универсальная программа для управления Bluetooth устройствами через браузер. Поддерживает HC-06, HC-05, HM-10, ESP32 и другие контроллеры и модули. Режимы: терминал, управление RC машинами, джойстик, умный дом. React + TypeScript.

Как добавить на GitHub:
-----------------------
1. Откройте страницу репозитория на GitHub
2. Нажмите на шестеренку ⚙️ рядом с "About" (справа)
3. Вставьте краткое описание в поле "Description"
4. Добавьте Website: https://alash-electronics.github.io/bluetoothWebApp/
5. Добавьте Topics (теги) из списка выше
6. Поставьте галочки:
   - ☑ Releases
   - ☑ Packages (если будете публиковать)
7. Нажмите "Save changes"
